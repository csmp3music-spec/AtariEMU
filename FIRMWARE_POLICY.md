# Firmware Policy

This project targets Atari systems from the XL/XE family through the Falcon.

## What this repository supports

- User-supplied original Atari ROMs, TOS images, and disk images
- Open-source replacement firmware where redistribution is clearly allowed
- User-supplied Atari MiNT and Linux/m68k boot media

## What this repository does not bundle

- Original Atari XL/XE OS ROMs
- Original Atari BASIC ROMs
- Original Atari TOS ROMs
- Atari DOS system disks

Those assets are modeled as user-supplied only.

Software becoming old, unsupported, or described as "abandonware" does not automatically make it public-domain for this repository.

## What can be bundled

- EmuTOS for ST, STE, TT, and Falcon-class machines

If EmuTOS binaries are added later, keep the license text with them and record the exact source version used.
