# ST / Falcon Focus

This is the recommended starting set for the Atari ST, TT, and Falcon side of the emulator.

## Best bundle candidates

These have the clearest redistribution story from project-maintained sources:

- EmuTOS
- TeraDesk
- NetSurf Atari
- AFROS

## Strong project-hosted candidates

These are useful and actively relevant, but you should review the exact license/release bundle before shipping them inside the app:

- FreeMiNT + XaAES
- fVDI
- QED
- TosWin2
- COPS
- HighWire

## Useful but mixed

- FreeMiNT OS Distro

This distribution is attractive for Falcon/TT users, but its own README says it ships only the demo version of HDDRIVER. Treat it as a reference distro or private convenience image, not as a clean default bundle.
