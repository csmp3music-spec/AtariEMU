# Curated Open Collection

This folder is a starter collection manifest for Atari software that appears openly redistributable from project-maintained sources.

It does **not** include old commercial Atari disks relabeled as "abandonware".

Use this folder as:

- A checklist for safe bundle candidates
- A place to drop downloaded open-source/public-domain packages you want the emulator to expose
- Documentation for what was verified and why

See `manifest.json` for machine targeting, source URLs, and notes.
See `ST_FALCON_FOCUS.md` for the recommended ST/TT/Falcon starter set.
